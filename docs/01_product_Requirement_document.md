# Product Requirement Document (PRD) - Craftdcode Portfolio

## 1. Project Overview
The objective is to create a high-end, premium portfolio website for **"Craftdcode"**, a team consisting of two developers and one designer. The website must showcase technical excellence and design sophistication through "technical elegance," monochrome aesthetics, and immersive parallax effects.

## 2. Goals & Objectives
- **Brand Identity**: Craftdcode - "Design · Develop · Deliver".
- **Visual Vibe**: Minimalist, high-performance, technical elegance.
- **Grid Motif**: Recurring 3x3 square grid from the logo as a design element.
- **Aesthetic Excellence**: Monochrome palette (#000000, #FFFFFF, #808080).
- **Technical Showcase**: GSAP-powered horizontal scrolling and parallax.

## 3. Tech Stack
- **Framework**: Next.js (App Router)
- **Smooth Scrolling**: Lenis
- **Animations**: GSAP + ScrollTrigger
- **Forms**: React Hook Form + Zod
- **Styling**: Vanilla CSS / CSS Modules

## 4. Key Features
- **Custom GSAP Loader**: Animating the 3x3 grid logo squares individually.
- **Hero Section**: Typography-heavy with 3D parallax background elements or blurred code snippets.
- **Horizontal Scroll Gallery**: GSAP ScrollTrigger powered showcase of portfolio items.
- **Services Section**: Design · Develop · Deliver workflow highlights.
- **Team Profiles**: Profile cards for 1 Designer and 2 Devs with magnetic/HOVER effects.
- **Minimalist Contact Form**: Form validation using Zod and React Hook Form.
- **Dynamic Case Studies**: Full-width parallax headers for project detail pages.

## 5. Visual Identity
- **Palette**: Monochrome (Deep Black, Pure White, Slate Grey).
- **The "Grid"**: 3x3 square grid motif throughout the site.
- **Typography**: High-end modern sans-serif.

## 6. Team Roles
- **Developer 1**: Core architecture, Next.js routing, and state management.
- **Developer 2**: Animation specialist, GSAP timelines, and scroll-jacking/Lenis integration.
- **Designer**: Visual assets, UI/UX layouts, and motion choreography.

## 7. Success Metrics
- Performance scores > 90 on Lighthouse.
- Average session duration high due to engaging animations.
- Positive feedback on "vibe" and "smoothness".
